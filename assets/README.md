# Assets Directory

This folder contains images and media for the README.

## Required Files

Add the following files to make the README display properly:

### Demo Video
- `demo.mp4` — A screen recording showing the gesture controller in action
- `thumbnail.png` (optional) — A thumbnail image if you want to link to YouTube instead

### Gesture Images (in `gestures/` subfolder)
- `ok_sign.png`
- `open_palm.png`
- `fist.png`
- `index_up.png`
- `l_sign.png`
- `peace_sign.png`
- `rock_sign.png`
- `call_me.png`
- `pinch_middle.png`

## Recommended Specs

- **Image size:** 200x200 pixels
- **Format:** PNG with transparent or clean background
- **Video size:** ~1280x720 or 1920x1080, MP4 format (H.264)
- **Video length:** 30-60 seconds showing key gestures

## Recording Tips

Use a tool like [OBS Studio](https://obsproject.com/), [ScreenToGif](https://www.screentogif.com/), or [ShareX](https://getsharex.com/) to record your demo.

> **GitHub Note:** GitHub doesn't natively play MP4 in README. Consider:
> 1. Converting to GIF for simple previews
> 2. Uploading to YouTube/Streamable and embedding a linked thumbnail
> 3. Using GitHub's video upload in Issues/Discussions and copying the URL
